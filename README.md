# 📊 India CPI Inflation Analysis (Excel Case Study)

## 📌 Project Overview
This project is an **Excel-based case study on India’s Consumer Price Index (CPI) inflation data**.  
The goal of this analysis is to understand inflation trends across months and years and derive meaningful insights using Excel formulas and structured data analysis.

---

## 🎯 Objectives
- Analyze **India CPI inflation trends**
- Perform **month-wise and year-wise comparisons**
- Apply Excel formulas for data analysis
- Identify **inflation patterns and key takeaways**
- Present results in a clean, structured, and understandable format

---

## 🛠 Tools & Technologies
- Microsoft Excel  
- Excel formulas (MIN, AVERAGE, percentage calculations, comparisons)
- Data structuring and formatting techniques

---

## 📂 Dataset
- Publicly available **India CPI Inflation data**
- Monthly CPI values across different periods  

> Dataset used strictly for learning and analytical practice.

---

## 🔍 Key Analysis Performed
- ✅ Month-wise CPI inflation analysis  
- ✅ Year-wise inflation comparison  
- ✅ Identification of lowest and highest inflation periods  
- ✅ Trend interpretation using Excel calculations  
- ✅ Well-structured data with readable formatting  

---

## 📈 Key Learnings
- Better understanding of **inflation and CPI indicators**
- Strengthened **Excel data analysis skills**
- Improved ability to interpret economic trends
- Learned best practices for building Excel case studies
